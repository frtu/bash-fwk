# Enable nvm & node

## Setup nvm

* ```enablelib dev-nvm```
* ```inst_nvm```
* ```nvmv``` => Get the current nvm version

## Install node

* ```nvminst 8|12``` => Install node 8 or 12
* ```nvmls``` => List all the installed env
